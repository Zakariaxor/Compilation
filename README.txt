Le rapport :
1-Classe main :
Pour commencer nous avons créé un jframecontenant 4  boutons charger qui permet de charger un fichier de type compila un bouton pour l’analyse lexical syntaxique sémantiqueayant chacun leurs propre classe .
2-Classe Les unités lexicauxréserves :
Dans cette classe on définie les nombre alphabets et les identificateurs .
3-Classe de l’analyse lexical :
Dans cette classe nous avons utilisés un extentd avec la classe Les unités lexicaux réserves qui permet de reconnaitre lexicalement un texte c’est-à-dire reconnaitre tout les mots de ce texte.
4-Classe de l’analyse syntaxique :
Dans cette classe nous avons utilisés un extentd avec la classe  lexical le but de cette analyse est de comparer avec la syntaxe  tout simplement elle permet de reconnaitre chaque mot ce qu’il veut dire .
5-Classe de l’analyse sémantique :
Dans cette classe nous avons utilisés un extentd avec la classe  syntaxique d’où elle permet d’afficher acoté de chaque ligne ou chaque instruction  ce qu’elle fait et se qu’elle veut dire , l’analyse sémantique c des déclarations.
